# chart-preview

A lightweight Angular 20 app for previewing [Vega-Lite v5](https://vega.github.io/vega-lite/) chart specifications. Paste a JSON spec on the left — the chart renders instantly on the right.

Used internally to test and validate chart specs generated by the Prismage data chat engine.

## Features

- Split-pane layout: JSON editor (left 25%) + live chart preview (right 75%)
- Auto-renders on paste/edit with 400 ms debounce
- Automatically scales bar charts so all rows fit in the visible area — no scroll
- Tooltip support on hover for all encoded fields
- SVG renderer with export action
- JSON parse and Vega-Lite render errors shown inline

## Running locally

```bash
cd chart-preview
npm install
npm start          # serves at http://localhost:8050
```

Or using the Angular CLI directly:

```bash
ng serve           # also serves at http://localhost:8050 (configured in angular.json)
```

## Building for production

```bash
ng build           # output in dist/chart-preview/browser/
```

## Usage

1. Start the dev server (`npm start`)
2. Open `http://localhost:8050`
3. Paste any valid Vega-Lite v5 JSON spec into the left panel
4. The chart renders automatically on the right

The app strips `width` and `height` from the pasted spec and computes dimensions from the container so the chart always fills the panel. For bar charts with a nominal y-axis, the step size per band is computed dynamically so all rows are visible within the window height.
